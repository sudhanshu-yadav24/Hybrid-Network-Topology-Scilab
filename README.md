# Hybrid Network Topology using Scilab

This project demonstrates a hybrid network topology created using Scilab.

## Topologies Used
- Star Topology
- Ring Topology
- Tree Topology

## Software Used
- Scilab 6.0.0

## Description
The network consists of 25 nodes. Node degrees, total nodes, and total edges are calculated and displayed.
